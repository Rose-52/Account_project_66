# 更新日志

## v1.0.0 - 2025-11-30

- ✅ **初始版本发布**
- 实现用户注册与登录（多用户隔离）
- 支持创建多个账户（如工资卡、零花钱）
- 账单管理：
  - 添加收入/支出（含金额、时间、备注）
  - 查看全部账单
  - 按日期范围查询
  - 关键词搜索（如“餐饮”、“工资”）
- 账单导出为 `.txt` 文件（自动按日期命名）
- 跨设备同步：
  - 服务端监听 9999 端口
  - 客户端通过 IP 上传/下载账本
- 数据持久化：MySQL 存储（`user` 表 + `account` 表）
- 使用 Maven 构建，JDBC 原生开发

> 📌 当前版本为课程期末作业提交版，功能完整，无已知严重 Bug。
